## 🎥 Demo

A demonstration of the Application Load Balancer distributing traffic across multiple EC2 instances.

🔗 **Watch Demo Video:**
[Click here to view the demo](https://drive.google.com/file/d/1zAFCGpXrc4F8vw1UBJUBo_Hl_SarZfWJ/view?usp=drive_link)

---

### 📌 What this demo shows:

* Traffic routed through **Application Load Balancer (ALB)**
* Requests distributed across **multiple EC2 instances**
* High availability across **multiple Availability Zones**
* Consistent application response from different instances

---

### 💡 Key Observation:

Refreshing the browser shows responses from different EC2 instances, proving that load balancing is working correctly.
